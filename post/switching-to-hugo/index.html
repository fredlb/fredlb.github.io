Switching to Hugo

Oct 28, 2014 · 1 minute read

I’ve now switched static site generator more times than I’ve written content with them. There wasn’t really anything wrong with Jekyll, Octopress or Pelican but in my strive for minimalism Hugo definitely strikes a chord with me. It also gives me a chance to check out Go, which is something I’ve been meaning to do for a few months.

Why is it so fun to learn a new, emerging programming language? Well, at least for me it probably stems from having been taught C/C++ and Java in an academic setting, where there isn’t a whole lot of space given to talk about the language itself. You probably shouldn’t even talk that much about language when learning programming anyway.

Currently I’m trying to write a simple raytracer in Go. Hopefully it will be a valuable lesson on goroutines :)
