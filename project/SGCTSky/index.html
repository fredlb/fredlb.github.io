SGCTSky

SGCTSky (SGCT - Simple Graphics Cluster Toolkit) is a sky simulation made for a local dome theatre in Norrköping, Sweden. It renders a sky (Preetham) and fake volumetric clouds to a cluster that then projects it onto the dome.

This was accomplished by developing a small application with SGCT. Most of the heavy processing is accomplished on the GPU with a simple GLSL fragment shader. Some precomputation is done prior to sending it to the GPU.

A small GUI to control the simulation was made in C#. It connects to the master node and parses all the parameters from the UI.
